position: static;

设置 position: static; 的元素不会以任何特殊方式定位；它是始终根据页面的正常流进行定位：

这个 div 元素设置 position: static;
------------------------

position: fixed;

设置 position: fixed; 的元素会相对视口定位，这意味着即使页面滚动也会停留在某个位置：

这个 div 元素设置 position: fixed;
-------------------------

position: relative;

设置 position: relative; 的元素相对于其正常位置进行定位：

这个 div 元素设置 position: relative;
--------------------------

position: absolute;

设置 position: absolute; 的元素会相对于最近的定位祖先进行定位（而不是相对于视口进行定位，比如 fixed）：

这个 div 元素设置 position: relative;
这个 div 元素设置 position: absolute;
